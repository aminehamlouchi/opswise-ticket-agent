## Steps

- Record the application name, version, error text, and exact workflow step.
- Check whether the problem follows the user, device, browser, or network.
- Confirm service status and known incidents before reinstalling software.
- Reproduce with a test account when permissions allow.

Escalation: Escalate to the application owner if data integrity, production access, or a known outage is involved.
